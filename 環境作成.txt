■angular
quickstartのダウンロード
https://github.com/angular/quickstart
解凍したフォルダをプロジェクト名にリネームする

■実行
コマンドプロンプト起動し、プロジェクトフォルダに移動

コマンドでプロジェクトに Node.js をインストール
npm install

コマンドで Node.js 標準の開発用Httpサーバーを起動
npm start

以上でプロジェクトフォルダ内の Src\index.htmlが起動される
